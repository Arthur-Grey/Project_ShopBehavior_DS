# Project_ShopBehavior_DS
В этом проекте представлен анализ датасета с использованием Python.

**Цель проекта** — показать навыки владения различными инструментами для 
обработки, анализа и визуализации данных

В рамках работы были выполнены следующие шаги:
- загрузка и первичный обзор датасета
- очистка данных 
- анализ распределений и взаимосвязей между признаками
- многоплановая визуализация различных категорий и графиков

Для работы использовались следующие инструменты:
- Python
- pandas
- numpy
- matplotlib / seaborn
- Yandex DataLens
- Git
- Google Colab
